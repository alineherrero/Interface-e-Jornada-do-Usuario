# Interface-e-Jornada-do-Usuario
Wireframe mobile de baixa fidelidade criado no Figma a partir de uma situação-problema real. O projeto foca na tarefa principal do usuário, jornada de uso, arquitetura da informação, clareza funcional e flexibilidade. Inclui fluxo navegável com dois caminhos distintos e validação por testes com usuários.

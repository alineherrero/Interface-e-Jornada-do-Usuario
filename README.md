🎓 Projeto de Interface — Site de Instituição Acadêmica

📌 Sobre o Projeto

Este projeto foi desenvolvido na plataforma Figma com o objetivo de criar o protótipo de um site para uma instituição acadêmica.

A proposta consiste em desenvolver uma interface moderna, organizada e de fácil navegação, facilitando o acesso dos alunos às principais informações do site.

💡 Ideia Principal

O projeto tem como foco a criação de um site intuitivo e funcional, onde o usuário consiga encontrar informações de forma rápida e sem dificuldades.

A estrutura foi pensada para garantir:

Clareza na navegação

Organização das informações

Facilidade de uso

🛠️ Ferramenta Utilizada
Figma — prototipação e design da interface

🎯 Resultados Esperados

Espera-se que o protótipo contribua para uma melhor experiência do usuário, permitindo que os alunos acessem o site da instituição de forma mais rápida, eficiente e prática.
